# E2E Тесты

E2E тесты для Electron Timer приложения, написанные с использованием Playwright.

## Установка

Перед запуском тестов необходимо установить зависимости:

```bash
npm install
```

Также нужно установить браузеры для Playwright:

```bash
npx playwright install
```

## Запуск тестов

### Все тесты

```bash
npm run test:e2e
```

### С UI режимом

```bash
npm run test:e2e:ui
```

### В headed режиме (с видимым браузером)

```bash
npm run test:e2e:headed
```

## Структура тестов

Тесты находятся в файле `timer.spec.ts` и покрывают следующие сценарии:

- Отображение начального состояния
- Запуск таймера
- Остановка таймера
- Пауза и возобновление таймера
- Изменение времени через кнопки input
- Изменение времени запущенного таймера
- Использование quick links
- Парсинг различных форматов времени
- Отображение статуса таймера
- Работа с большими значениями времени

## Примечания

Тесты запускают Electron приложение через webServer в конфигурации Playwright. 
Приложение открывается как локальный файл (`file://` протокол).

Для работы тестов необходимо, чтобы приложение было собрано (`npm run build`).
